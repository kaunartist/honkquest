## Description

HonkQuest, built on NestJS & Typescript.

## Installation

```bash
$ nvm install 16.10.0
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

```bash
# e2e tests
$ yarn test:e2e
```

Insomnia/Postman collection can be imported from honk.har
